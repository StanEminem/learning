# How to start Dcc Demo
### install dependency
```sh
$ npm install
$ cd public
$ bower install
$ cd ../
```
### start Dcc Demo
```sh
$ npm start (npm start &)
```

# How to test Dcc Demo
```sh
$ npm test
```
